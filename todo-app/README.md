### Built With

- CSS
- JS/Vue

## Features

- Add new task
- Edit task  
- Complete the task
- Delete any task
- Delete all completed in once
- Sort tasks (All/Active/Completed)
- Tasks are saved into localStorage  
- The app is fully adaptive
